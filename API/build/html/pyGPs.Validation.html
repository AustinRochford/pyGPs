pyGPs.Validation.valid.k_fold_validation(x, y, K=10, randomise=False)[source]¶

Generates K (training, validation) pairs from the items in X. The validation iterables are a partition of X, and each validation iterable is of length len(X)/K. Each training iterable is the complement (within X) of the validation iterable, and so each training iterable is of length (K-1)*len(X)/K.

Parameters:
  • x – training data
  • y – training targets
  • K – number of folds
  • randomise – boolean flag. Shuffle data first if it is true.s
